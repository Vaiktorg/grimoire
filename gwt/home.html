About GWT

GWT is a set of tools that allows web developers to create and maintain complex JavaScript front-end applications in Go. Instead of writing the front-end code in JavaScript, developers can write in Go, which is then cross-compiled to highly optimized JavaScript that runs across all browsers.

Features

  • Write front-end in Go instead of JavaScript
  • Cross-compile Go to optimized JavaScript
  • Strong typing and tooling of Go
